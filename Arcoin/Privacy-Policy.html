1. Data Collection

The bot collects user data including User IDs, Nicknames, transaction logs, and command interactions for the purpose of managing virtual currency balances and transactions.

User data is used to manage virtual currency, store transaction logs, and process user interactions within the bot’s system.

All collected data is stored indefinitely unless requested for removal. Data does not automatically delete when a user leaves a server or deletes their account.

4. Data Sharing

The bot does not share or sell any user data to third parties. All data remains securely stored and used solely for the bot’s functionality.

5. Security

While efforts are made to ensure the security of user data, the developer cannot guarantee complete security.

6. User Rights

Users can request to view or remove their data by contacting the bot owner via Discord.
